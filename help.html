Приложение для пациентов с диагностированным диабетом

Было разработано удобное мобильное приложение(Android, IOS), которое поможет пользователям контролировать состояние здоровья (сахар, артериальное давление, гемоглобин), отслеживать процесс лечения и приема лекарств, за счет отслеживания состояния пациентов и хранения медицинских данных для использования в научных целях.

Технологии

ASP.net, C #, MVC, Microsoft SQL server, Java

Приложение позволяет пациенту и лечащему врачу успешно выстроить схему лечения. Своевременный контроль показателей сахара, давления и гликированного гемоглобина поможет вовремя и качественно изменить схему лечения для достижения наилучшего результата лечения.

Также есть веб-часть врача, где врач может назначить пациенту соответствующее лечение (в зависимости от состояния пациента). Врач сможет следить за динамикой лечения пациента и видеть аналитику приема лекарств и снижения уровня сахара в крови.

Основная цель

Для пациента вовремя принимать препараты для лучшего эффекта терапии

Для врача увеличение эффективности лечения пациента, контроль состояния пациента

Ключевые особенности приложения:

схему лечения назначает лечащий врач

push-уведомления напомнят о приеме лекарств

журнал пропущенных и подтвержденных приемов лекарств доступен через календарь

данные по показателям сахара, давления выводятся в виде графиков

сохраняется история приема лекарств, показателей сахара, давления и гемоглобина

ввести показатели давления, сахара, гемоглобина и приема лекарства возможно исторически за пропущенные дни

кнопка быстрого вызова, позволит мгновенно позвонить доверенному лицу в экстренном случае

регистрация в приложении по смс-коду обеспечивает защиту персональных данных

приложение доступно для бесплатного использования
