    • Приложение для водителей Android и iOS

    Devcloud Solutions разработает систему мобильных приложений Android и iOS для водителей, которая позволит держать свой баланс под полным контролем и снять деньги в любое удобное время

    Приложение сделает вывод денег с баланса профиля водителя яндекса, на банковскую карточку водителя быстрее, удобнее и увеличит клиентский поток

    Характеристика сервиса. Сервис состоит из модулей — Сервер API и Базы данных

    Мобильное приложение построена на технологии Flutter, административная часть системы построена на мгновенно работающей технологии Vue.JS

    Реализована административная панель для Директора подключившегося таксопарка, Менеджера, Бухгалтера

    Ключевые модули приложения

    • Авторизация, Регистрация,
      Смена пароля

    • Подтверждение СМС, Пин-код для
      дальнейшего входа

    • Главная страница с данными профиля и
      балансом, возможность выбора карты и
      указания суммы перевода

    • История переводов

    • Новости

    • Настройки
